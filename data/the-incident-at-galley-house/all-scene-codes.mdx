<h2 id="how-to-read">
  How to read and enter a scene code
</h2>

Every row separates the same three fields:

- **Time** is the numbered Timeline slot.
- **Location** is the two-letter machine abbreviation.
- **People** includes every profile physically present, even when someone is
  silent or already dead.

For example:

```text
03-LI-1-4-5-6-7-8-9
```

means:

- Time 03;
- Living Room (`LI`);
- People 1, 4, 5, 6, 7, 8, and 9.

I do not need to rename every profile before using the list. The machine accepts
the numeric or letter ID as long as the person and location have been added.

<h2 id="code-not-working">
  What to check when a listed code does not work
</h2>

A code from the tables can still be unavailable if the machine setup is
incomplete.

I check:

1. the required location has been added under Locations;
2. every required numeric or letter profile has been calibrated;
3. the Admin Card is inserted for the modern scenario;
4. ID 64 has been revealed before attempting Ervin's scenes;
5. the code uses every participant shown in the row;
6. the Time 00 photograph has been scanned before entering Scene 00.

The code itself does not automatically create a missing room or profile.

<h2 id="faq">
  The Incident at Galley House All Scene Codes FAQ
</h2>

### Can I enter the scene codes out of story order?

Yes, once the required time, room, and profiles are available. A code cannot
unlock a location, hidden profile, Admin ID, or photograph that the machine has
not registered yet.

### Do character names matter when entering a code?

No. The machine validates the numeric or letter IDs. I still rename confirmed
profiles because the transcript and Timeline then display useful names, but a
wrong label does not alter the underlying code and can be corrected later.

### Do I need Person 12 and Person K before entering the modern scenario?

I can continue after the main 1–11 route, but the original case is not truly
complete until both hidden profiles are reconstructed. They also unlock
separate achievements, so I resolve them before using the completed save for
cleanup.

### How many valid reconstructions are listed here?

There are 106: 88 ordinary original scenes, two hidden-victim scenes, 15 valid
modern scenes, and Scene 00. The two achievement-only inputs are shown
separately and are not included in that total.

### Can I return after the ending for a missed code?

Yes. The completed save remains available for optional-scene and achievement
cleanup, so I do not restart solely for one missing reconstruction.

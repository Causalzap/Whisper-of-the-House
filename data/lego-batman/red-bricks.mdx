<h2 id="best-red-bricks-to-get-first">
  Best Red Bricks To Get First
</h2>

Red Bricks in LEGO Batman Legacy are mostly cosmetic modifiers, so there is no gameplay power pick like an old-school Stud Multiplier.

If you are cleaning them up efficiently, prioritize them like this:

1. **Filthy Rich** — grab this early if you are replaying Carmine Falcone, because the safe code is simple: **596**.
2. **Bats** — do this once you have Nightwing and Batgirl available, because the Wayne Tower garbage truck puzzle needs both.
3. **Mission Red Bricks you are already near** — if you are replaying a level for collectibles, grab its Red Brick in the same run.
4. **Cheap Bat-Mite’s Shop Red Bricks** — buy shop Red Bricks whenever they appear, starting with cheaper ones like Construction, Sporty, Glitch, Groovy, Festive, and Futuristic.
5. **Beauty last if studs are tight** — it costs 50,000 studs, so it is fine to buy later if you are still short on currency.

The important thing is not power level. It is avoiding wasted replay time. Clear mission Red Bricks while you are already in those levels, then refresh Bat-Mite’s Shop for the remaining shop-only Red Bricks.

<h2 id="why-bat-mites-shop-shows-suits">
  Why Bat-Mite’s Shop Shows Suits
</h2>

Bat-Mite’s Shop is a mixed reward shop. That is why you may see more purchasable items than the 9 shop Red Bricks.

The Red Brick total is still **23**:

- **14** Red Bricks are hidden in main story levels.
- **9** Red Bricks are bought from Bat-Mite’s Shop.
- Suits are separate rewards and do not count as Red Bricks.

<h3 id="what-do-red-bricks-do">
  What do Red Bricks do in LEGO Batman Legacy?
</h3>

Red Bricks unlock cosmetic modifiers for suits and vehicles. They change colors or paint styles instead of giving old LEGO-style cheat effects.

<h3 id="is-filthy-rich-a-stud-multiplier">
  Is Filthy Rich a Stud Multiplier?
</h3>

No. In LEGO Batman Legacy, Filthy Rich is a green color modifier with darker green highlights. It is not a Stud Multiplier.

<h3 id="what-is-the-carmine-falcone-red-brick-code">
  What is the Carmine Falcone Red Brick code?
</h3>

The Iceberg Lounge vault code is **596**. Enter 596 to get the **Filthy Rich** Red Brick.

<h3 id="why-does-bat-mites-shop-show-suits">
  Why does Bat-Mite’s Shop show suits instead of Red Bricks?
</h3>

Bat-Mite’s Shop has a rotating item pool. It can show suits and Red Bricks. Only items labeled **Red Brick** count toward Red Brick completion.

<h3 id="how-do-you-refresh-bat-mites-shop">
  How do you refresh Bat-Mite’s Shop?
</h3>

Leave Bat-Mite’s Shop and re-enter. The available items refresh when you come back.

<h3 id="do-suits-count-as-red-bricks">
  Do suits count as Red Bricks?
</h3>

No. Suits are separate shop rewards and do not count toward the 23 Red Brick total.

<h3 id="which-red-bricks-are-in-bat-mites-shop">
  Which Red Bricks are in Bat-Mite’s Shop?
</h3>

The shop Red Bricks are **Construction, Sporty, Glitch, Groovy, Festive, Futuristic, GCPD, Magic, and Beauty**.

<h3 id="what-should-you-do-if-a-red-brick-is-missing">
  What should you do if a Red Brick is missing?
</h3>

Check the mission Red Bricks first, then refresh Bat-Mite’s Shop until the remaining shop Red Brick appears.

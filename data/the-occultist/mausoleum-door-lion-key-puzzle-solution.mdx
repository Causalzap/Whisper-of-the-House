<h2 id="where-is-the-mausoleum-door-lion-key-puzzle">
  Where is the Mausoleum Door Lion Key Puzzle in The Occultist?
</h2>

The **Lion Key Path Puzzle** is on the **Mausoleum door in Godstone Cemetery**.

This is the cemetery gate puzzle that blocks the route forward. It is not just a decorative lock. It is the progression gate that opens the underground path leading deeper into the game.

### What you need before this puzzle
- the **Lion Key**
- the **three Cemetery Medallions** that show path fragments
- the journal clue that lets you understand the final combined route

<GuideCallout title="Important requirement">
If the route looks impossible or incomplete, the usual problem is that you have not collected all **three Cemetery path Medallions** yet.

This puzzle is meant to be solved by **combining multiple route clues**, not by guessing on the door itself.
</GuideCallout>

<h2 id="mausoleum-door-lion-key-puzzle-solution">
  Mausoleum Door Lion Key Puzzle solution
</h2>

The solution is:

- combine the **three Cemetery Medallion path clues**
- reconstruct the **full final route**
- use the **Lion Key** to trace that path on the Mausoleum board
- draw it **continuously from start to finish**

That is the full answer.

This is not a symbol match puzzle and not a “press all correct nodes in any order” puzzle. It is a **single-route tracing puzzle**.

### How to solve it step by step
1. Collect the **three Cemetery Medallions** that show the route fragments
2. Check the journal clue that combines those path pieces
3. Go to the **Mausoleum door**
4. Use the **Lion Key**
5. Start from the correct entry point
6. Trace the **entire combined route** without breaking it
7. Finish at the correct end point
8. Wait for the **Mausoleum door** to open

<h2 id="how-the-medallion-clue-works">
  How the Medallion clue works
</h2>

This puzzle works by taking **three separate route fragments** and turning them into **one continuous line**.

The clue is not asking you to choose one Medallion pattern. It is asking you to understand how the three Cemetery path Medallions overlap into a final route.

That is why this puzzle can feel confusing at first. Players often assume each Medallion is its own answer, when the real answer is the **combined route**.

### The short logic
- the Medallions are **partial clues**
- the journal shows how they fit together
- the Mausoleum board expects the **final merged path**

<GuideCallout title="Common mistake">
The biggest mistake here is tracing only **one** Medallion pattern.

That will not work.

The door only opens when you draw the **full combined route**.
</GuideCallout>

<h2 id="does-the-mausoleum-door-puzzle-have-an-order">
  Does the Mausoleum Door puzzle have an order?
</h2>

Yes. This puzzle **does have an order**.

You must trace the route:

- from the **correct starting point**
- in **one continuous motion**
- all the way to the **correct end point**

This is not a puzzle where the right shape alone is enough. The route has to be drawn as a valid path.

<GuideCallout title="Important order rule">
Even if your drawn shape looks correct, the puzzle can still fail if:

- you started from the wrong node
- you broke the route midway
- you reversed the intended direction
</GuideCallout>

  What do you get from the Mausoleum Door puzzle?
</h2>

Solving the Mausoleum door puzzle opens the **Mausoleum entrance**.

From there, you move into:

- the **underground maze route**

So this puzzle is not just a collectible side interaction. It is one of the most important progression locks in the cemetery section.

If you are still missing the required path Medallions, use the <Link href="/the-occultist/all-medallion-locations/">All Medallion Locations guide</Link> before you come back here.

<h2 id="why-players-get-stuck-on-the-mausoleum-door-puzzle">
  Why players get stuck on the Mausoleum Door puzzle
</h2>

This puzzle usually stops players for one of four reasons:

- they are missing one of the **three Cemetery path Medallions**
- they try to trace only one Medallion route instead of the combined path
- they start from the wrong point
- they do not keep the line continuous from start to finish

### The short logic
- collect all three route clues
- merge them into one pattern
- trace that path correctly with the Lion Key

This is one of those puzzles where the answer is clear once the clue is understood, but very frustrating if the player treats each fragment as a separate final solution.

<h2 id="what-to-do-if-the-mausoleum-door-puzzle-is-not-working">
  What to do if the Mausoleum Door puzzle is not working
</h2>

If the Mausoleum door is not opening, check these first:

1. **Make sure you have all 3 Cemetery path Medallions**
2. **Re-check the combined journal route**
3. **Make sure you are tracing the full route, not one fragment**
4. **Start from the correct point**
5. **Keep the line continuous**
6. **Finish at the correct endpoint**

If the door still does not open, the most likely issue is not a hidden mechanic. It is almost always one of these three problems:

- a missing Medallion clue
- the wrong start point
- a broken or reversed route

<GuideCallout title="Best troubleshooting rule">
If the Mausoleum puzzle feels wrong, stop redrawing random paths.

Go back to the clue logic first:

**3 Medallion route fragments → 1 full continuous path**
</GuideCallout>

<h2 id="what-is-shifting-sands">
  What is Shifting Sands?
</h2>

**Shifting Sands** is the major new map and progression layer added to **Soulmask 1.0**.

The cleanest way to describe it is:

**a full new region with its own travel rhythm, ship progression, collectible routes, boss flow, and base-building pressures.**

What makes it different from just “more map” is that it changes how players think about:

- traversal
- mobility
- map transfer
- early-game setup
- tribesmen workload
- ship-based progression

For most players, the first real Shifting Sands questions are not lore questions.

They are practical ones:

- Where do I start?
- Do I need a new save?
- Can I move between maps?
- When does the ship actually become useful?
- How early should I start building a real base?

That is exactly what this hub is built around.

<h2 id="quick-start-faq">
  Quick Start FAQ
</h2>

### Can new players start directly in Shifting Sands?

Yes. New players can go directly into Shifting Sands rather than being forced to fully clear the old map first.

### Do I need a new save for Shifting Sands?

In **single-player**, you are starting a **new map** and using **save inheritance** if needed.  
On **servers**, the system works through transfer gates instead.

### Can I use my old save or existing character?

Yes, but the answer depends on mode.  
Single-player uses inherited character data.  
Servers use Cross-Map transfer.

### Can I bring ships back to Cloud Mist Forest?

No. Ships and related ship systems are one of the clearest Shifting Sands-only mechanics.

### Which guide should I read first?

If you are new, start with the <Link href="/soulmask/shifting-sands-beginner-guide">Beginner Guide</Link>.  
If your question is about transfer rules, start with the <Link href="/soulmask/cross-map-transfer-guide">Cross-Map Transfer Guide</Link>.

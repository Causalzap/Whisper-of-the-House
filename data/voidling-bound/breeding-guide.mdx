<GuideQuickAnswer title="Quick Answer">
In **Voidling Bound**, breeding is how you build stronger same-species offspring through the **Pheromonal Nest**. Use two Voidlings of the same species, check the offspring stat range, then complete a mission to hatch the egg.

The most important rules are:

- early offspring inherit reduced parent values until Henry's Hatchery upgrades improve inheritance;
- an upgrade that says **80%** means a percentage of the parent's attribute value, not 80 stat points;
- breeding can pass up to **80 inherited base attribute points**, while the final 20 points come from leveling;
- Dual Natures come from breeding different Nature types;
- same-subspecies parents preserve a form more cleanly;
- different subspecies can fall back to a shared evolution point;
- Splicing should usually happen after the breeding line is ready.
</GuideQuickAnswer>

<h2 id="how-breeding-works">
  How breeding works
</h2>

Breeding uses the **Pheromonal Nest**. Pick two Voidlings from the same species, check the possible offspring stat range, start breeding, then complete a mission to hatch the egg.

<GuideTable
  caption="Voidling Bound breeding basics."
  columns={basicsColumns}
  rows={basicsRows}
/>

The key detail is the stat range. If the nest shows **01–20** for Strength and Essence, the child can roll up to 20 in those stats. If a stat only shows a low upper range, that parent pair is not good enough for that stat yet.

<h2 id="inheritance-penalty-and-cap">
  50% inheritance penalty, Hatchery percentages, and the inherited point cap
</h2>

This is the part that explains most confusing breeding results.

Before the right Hatchery upgrades, the child does not simply copy the best parent values. The inherited value is reduced, which is why two strong parents can still produce a weaker child.

The long-term goal is to push inheritance closer to the full parent value through Henry's upgrades.

<GuideTable
  caption="Important breeding stat limits."
  columns={capColumns}
  rows={capRows}
/>

This is separate from percentage-based Hatchery upgrades. An upgrade that says **80%** refers to how much of a parent's attribute value can pass down. The **80-point cap** refers to the total inherited base attribute points the offspring can carry before level-up allocation.

The practical build target is:

<strong>80 inherited base points from breeding + 20 level-up points = a final 100-point base stat line.</strong>

That is how a final level 20 Voidling can reach 20 in all five base attributes. Nature bonuses and monster part bonuses are separate layers.

<h2 id="faq">
  Voidling Bound breeding FAQ
</h2>

### How do you breed Voidlings in Voidling Bound?

Use the Pheromonal Nest, choose two Voidlings from the same species, check the possible offspring stat range, then start breeding. Complete a mission to hatch the egg and inspect the result.

### Can you breed different species?

No. Breeding is a same-species system. Keep useful duplicates from the species you want to build.

### Why are my offspring stats lower than the parents?

Early breeding has reduced inheritance. Henry's Hatchery upgrades improve how much of a parent's attribute value can pass down.

### Does 80% mean 80 points?

No. An upgrade that says 80% means 80% of a parent's attribute value can pass down. The 80-point cap is a separate inherited base point limit before level-up allocation.

### What is the 80-point breeding cap?

Breeding can pass up to 80 inherited base attribute points. The remaining 20 points come from leveling the final child, which is how a final build can reach 20 in all five base stats.

### How do I get 20 in multiple stats?

Train parent candidates to level 20, max one target stat on each parent, upgrade Henry's Hatchery, then breed parents that cover different stats. Keep children that inherit multiple strong stats and use them as the next parent line.

### Is 20 the max base stat?

20 is the target base value for each attribute. Nature bonuses and monster part bonuses can push displayed values higher, but the breeding route is focused on building the base stat line.

### How do eggs hatch?

After you start breeding, complete a mission. Return to the ship and inspect the egg at the incubation station.

### How do you get Dual Natures?

Breed two same-species Voidlings with different Natures. Dual Nature rarity follows the lower rarity between the parents, so use high-tier parents if you want a high-tier Dual Nature.

### What are the Dual Nature combinations?

Strong + Witty creates Ferocious, Strong + Healthy creates Robust, Strong + Tenacious creates Enduring, Strong + Agile creates Athletic, Witty + Healthy creates Lively, Witty + Tenacious creates Resourceful, Witty + Agile creates Elegant, Healthy + Tenacious creates Rugged, Healthy + Agile creates Spry, and Tenacious + Agile creates Relentless.

### Can Nature rarity and Nature type inherit separately?

Yes. Nature planning has two parts: the Nature type you want and the rarity tier you want. A parent with a useful high-tier Nature can still produce a lower-tier outcome if paired with a lower-rarity parent.

### How does subspecies inheritance work?

If both parents share the same subspecies, the child keeps that subspecies. If the parents are different subspecies, the child can return to the closest shared evolution point between the two parent lines.

### Can Golden Egg variants pass through breeding?

Golden Egg variants can have value for breeding and Splicing, and special variant traits may be worth preserving. Inspect them before releasing or using them as casual parents.

### Can spliced Voidlings breed?

Spliced Voidlings normally cannot breed through the standard system. A late Spliced Breeding upgrade can open a specialized path, but the safer rule is to finish breeding before Splicing.

### How do I unlock more Henry upgrades?

Raise Wrangler Rank by discovering and documenting new genes through missions and evolution paths. Higher rank unlocks more Hatchery upgrades from Henry.

### Should I breed before Splicing?

Yes for normal progression. Breed the base stats, Nature, and subspecies line first. Splicing should be the final custom-build step unless you already have the late Spliced Breeding upgrade and know exactly why you are using it.

### Is breeding required for the main story?

Perfect breeding is not required for basic story progress, but breeding becomes important if you want stronger parent lines, Dual Natures, better stats, late-game builds, and cleaner Splicing candidates.

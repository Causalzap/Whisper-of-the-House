<GuideQuickAnswer title="Quick Answer">
To unlock corporation access in **SpaceCraft**, finish the first FTL route, reach **RakMu**, talk to the **Station Clerk**, then start **Instructor Ingrid Alvares' Noviciate test**. The test has two parts: a cognitive door puzzle where you use **handprint panels** to unlock every required door, and a physical route where you crouch, climb a ladder, and jump across platforms to reach the exit.

After the assessment is complete, you can create or join a corpo. If the option is still locked, finish any remaining early onboarding, refined-ore sale, blueprint access, or Knowledge-menu progression before checking the corpo interface again. 
</GuideQuickAnswer>

</h2>

<h2 id="noviciate-test-walkthrough">
  Noviciate test walkthrough: handprint panels, ladder, and platforms
</h2>

The Noviciate test is short, but the important details are easy to miss. The first half is a **cognitive test** built around handprint panels and locked doors. The second half is a **physical test** built around crouching, climbing, and jumping.

During the cognitive test, do not simply run forward and wait for the route to solve itself. Check each room for **handprint panels**. If a door remains locked, backtrack and look for another panel you did not use.

During the physical test, hold crouch when prompted, then keep moving until you reach the final climb. Near the end, the route continues upward: climb the ladder, jump across the platforms, and follow the exit path.

<h2 id="first-corpo-request">
  First corpo request: start the corporation loop
</h2>

Creating or joining a corpo unlocks access. The actual corporation loop starts with the first corpo request.

In the captured build, the first request route used a **Module Kit Sample**. That matters because Module Kits and related parts are also used by ship upgrades, FTL chains, and early factory progression.

<figure className="my-8">
  <Image

  <figcaption className="mt-3 text-sm text-slate-400">

Before feeding materials into a corpo request, check whether those same parts are needed for your ship, cargo hold, or base. A request can be worth doing, but it can also delay your immediate progression if you give away your last module parts too early.

If cargo is already the limiting factor, buy **Cargo Accommodation** from the Knowledge menu when it is available. It unlocks **Small Cargo Hold**, which makes mining trips and request deliveries much less painful. For recipes and installation steps, use the <Link href="/spacecraft/ship-upgrades-cargo-inventory">SpaceCraft Ship Upgrades Guide</Link>.

<h2 id="when-wholesale-contracts-unlock">
  When do wholesale contracts unlock?
</h2>

Wholesale contracts appear after the first corporation request steps in the captured build. The game presents them as longer-term, more demanding production contracts from the corporation desk.

If you do not see them yet, do this first:

<ol className="my-6 list-decimal space-y-2 pl-6">
  <li>Finish Noviciate.</li>
  <li>Create or join a corpo.</li>
  <li>Complete the first corpo request.</li>
  <li>Open the corpo interface and check Licenses.</li>
  <li>Return to the corporation desk and check available requests again.</li>
</ol>

Do not rush wholesale contracts before your ship cargo and base output can support them. In the captured build, a later wholesale request could ask for hundreds of ingots, which turns it into a base-production problem rather than a simple hand-in.

<h2 id="faq">
  SpaceCraft Noviciate and corporation FAQ
</h2>

### What is Noviciate in SpaceCraft?

Noviciate is the rank route that unlocks corporation access. Talk to Ingrid at RakMu and pass the cognitive and physical assessment.

### How do I pass the cognitive test?

Use the handprint panels to unlock all required doors. If a door stays locked, backtrack and find the missing panel.

### How do I pass the physical test?

Hold crouch when prompted, then continue to the final ladder and platform jumps. The exit route is upward near the end.

### Why can't I join or fund a corpo?

Finish Ingrid's Noviciate test first. If the option is still locked, finish remaining early onboarding, refined-ore sale, blueprint access, or Knowledge-route steps.

### Is Novician the same as Novvician?

In Early Access, you may see Novician, Novvician, or Noviciate-related wording in different places. Treat them as the same corporation-access rank route unless a later patch changes the rank system.

### Should I create a solo corpo?

Yes, if you want control. The confirmed early pressure is not a huge startup material cost; it is the ongoing need for corpo credits, request materials, license choices, and production.

### What should I do right after unlocking corpo access?

Create or join a corpo, complete the first corpo request, and strongly consider Cargo Accommodation / Small Cargo Hold if cargo is already slowing down your deliveries.

### Are corpo permits the same as Knowledge permits?

No. Knowledge permits are personal unlocks like Modular Systems, Electromagnetism, Cargo Accommodation, Personal Factory, and Automated Factories. Corpo permits are corporation-side upgrades tied to corpo credits, licenses, requests, shared ships, shared bases, and storage.

### When do wholesale contracts appear?

They appear after early corporation request progression in the captured build. Finish the first corpo request, check Licenses / Corpo Permits, then return to the corporation desk.

<h2 id="base-building-automation-guide">
  SpaceCraft base building and automation guide
</h2>

Base building in SpaceCraft is not only a crafting menu. It is where the game starts turning into a production and logistics game.

Your first base has one job:

**turn a nearby resource deposit into useful ingots without making you carry every item by hand.**

The clean first-base route is:

<ul className="my-6 list-disc space-y-2 pl-6">
  <li>unlock Personal Factory,</li>
  <li>get the Base Core Drive and Construction Tool,</li>
  <li>find an unclaimed resource deposit,</li>
  <li>place the Base Core Drive close enough to that deposit,</li>
  <li>build an Extractor directly on the deposit,</li>
  <li>place a Smelter nearby,</li>
  <li>connect power with a Pylon,</li>
  <li>select the Smelter input,</li>
  <li>then craft a Drone and create a Drone Route.</li>
</ul>

This guide focuses on the first reliable automation loop. It does not pretend to be an endgame mega-factory layout.

If you are still before FTL, read the <Link href="/spacecraft/scanlyzer-alpha-ftl-engine">Scanalyzer Alpha and FTL guide</Link>. If you need resources for base parts, use the <Link href="/spacecraft/resources-locations">Resources Guide</Link>.

</h2>

Base building starts once **Personal Factory** and the first construction items become available.

The first items to understand are:

<ul className="my-6 list-disc space-y-2 pl-6">

<h2 id="find-unclaimed-resource-deposit">
  How to find an unclaimed resource deposit
</h2>

Because SpaceCraft is an online game, the easiest deposits near busy starter routes may already be claimed or crowded. If you try to place a base too close to another player's setup, you can waste time wondering why your plan does not work.

Before placing the Base Core Drive, use a quick scan flow instead of trusting the first marker you see. The goal is to confirm three things: the deposit is real, the site is not already claimed, and the base can still fit a compact Extractor, Smelter, and Pylon setup.

<h2 id="power-sources">
  Power sources: Base Core, Solar Plant, and backup options
</h2>

Your first power source is the **Base Core Drive**, but the base will eventually need more power if you keep adding machines and routes.

The most important confirmed early power expansion is the **Solar Plant**. It generates electricity without fuel, but its description notes that it takes space and must be oriented correctly.

That means you should place solar in open space and rotate or confirm placement carefully. Do not squeeze it into a cramped, blocked, or awkward spot and then assume the power system is broken.

<h2 id="extractor-smelter-pylon">
  Build the first Extractor, Smelter, and Pylon loop
</h2>

The first working loop is simple:

**deposit → Extractor → Smelter → ingots**

Place the **Extractor** directly on the deposit. Then place the **Smelter** close enough to keep the first layout compact. Use a **Pylon** and cable connection to connect the machines to the base power grid.

This is the first point where the base becomes useful. Do not add Warehouse, Assembler, Command Tower, or Landing Pad before this basic loop works.

<h2 id="smelter-input">
  Select the Smelter input
</h2>

A Smelter can be placed, powered, and still do nothing if you never select the input.

Open the Smelter and choose the correct ore input, such as **Iron Ore**. In the shown setup, the Iron Ingot recipe needs Iron Ore, so the Smelter input must match the ore your Extractor is producing.

If the Smelter is idle, check the input before rebuilding the entire base.

<h2 id="drone-route">
  Create your first Drone Route
</h2>

After the first machine loop is ready, the next step is logistics.

If the route tool says **requires a Drone**, it means you need to craft a Drone before the route can run. This is not a bug in the route system.

The basic drone flow is:

<h2 id="automation-complete">
  When is the first automation setup complete?
</h2>

Your first automation setup is complete when the base can do these things:

<ul className="my-6 list-disc space-y-2 pl-6">
  <li>hold a claimed build area,</li>
  <li>power one Extractor and one Smelter,</li>
  <li>turn one deposit into ingots,</li>
  <li>move materials through at least one Drone Route,</li>
  <li>and still have enough footprint and energy left for the next upgrade.</li>
</ul>

That last point is important. A setup is not really stable if it only works while maxing out footprint or power. Once the first route works, expand based on the next bottleneck: add power if energy is tight, add Warehouse if storage is messy, add Assembler when you need industrial compounds, and use Command Tower when footprint becomes the real limit.

<h2 id="faq">
  SpaceCraft base building FAQ
</h2>

### When can I start base building in SpaceCraft?

Base building starts once Personal Factory and the first base construction items become available. Your first key items are Construction Tool, Base Core Drive, Extractor, Smelter, Pylon, Cable, and Base Deploy Kit.

### What does the Base Core Drive do?

The Base Core Drive claims the surrounding area and supplies electric power to the base. In the shown build, it appears in the Company Exchange for 499.99 credits, but Early Access values can change.

### Where should I build my first base?

Build near an unclaimed Iron or Copper deposit. These resources feed the most useful early crafting and automation chains.

### How do I find an unclaimed resource point?

Scan from orbit, open the map, fly low over the marker, check whether another player has already claimed or built around the area, then place your Base Core Drive only after confirming the deposit is usable.

### Should I build around Quartz?

Quartz is useful, especially for Crystalline Focuser and later upgrade paths, but it is usually better as a second-stage base target after Iron or Copper production is stable.

### Should I build around Gravite?

Not for your first base. Gravite is more useful as a special scouting or progression resource than as the anchor for your first ore-to-ingot loop.

### Why can’t I place an Extractor?

Extractors must be placed directly over a valid resource deposit. If the deposit is already claimed or the blueprint is not aligned with the deposit, placement may fail.

### Why is my Smelter not working?

Check power, input, and material flow. The most common issue is forgetting to select the correct Smelter input, such as Iron Ore.

### How do I connect buildings to power?

Use a Pylon and cable connection to link the building back to the base power grid. Then check Energy Consumption in the base terminal.

### How does Solar Plant orientation work?

The Solar Plant description says it needs correct orientation and space. Place it in an open area, rotate it carefully during placement, and avoid cramped or blocked spots.

### What should I do if base power consumption is too high?

Check the base terminal. If Energy Consumption is close to the limit, add more power before adding more Extractors, Smelters, Warehouses, or routes.

### Does Drone Route have a range limit?

The exact range can vary by setup and current build, so do not assume one Drone can solve every distance problem. If a route fails or feels inefficient, shorten the route, add storage, or reorganize buildings before scaling.

### Why does Add Route require a Drone?

The route tool needs an actual Drone before the route can run. Craft a Drone first, then create the route again.

### Can I unclaim or move a base?

Do not rely on moving a base as your main plan unless you have confirmed the current patch’s base removal or relocation option. Before placing the Base Core Drive, assume the site choice matters and scout carefully.

### Is Trading Box useful solo?

Not early. Trading Box is mainly useful for corporation or co-op sharing. Solo players should focus first on Extractor, Smelter, Pylon, power, and Drone Routes.

### What should co-op players do differently?

Split roles. One player handles layout, one handles automation, one scouts unclaimed deposits, and one moves missing materials. This is cleaner than everyone mining randomly.

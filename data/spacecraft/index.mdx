<h2 id="beginner-route">
  Beginner route: Babylon to RakMu
</h2>

The first route is about escaping the starter loop cleanly. Your job is to mine Copper and Iron, return to Babylon, craft and install Scanalyzer Alpha, use it on Solar Alpha 2, finish the FTL chain, refill Magnetic Plasma, and reach RakMu.

The most important beginner rule is simple: **crafting a module is not the same as installing it**. If Scanalyzer Alpha, a detector, a cargo hold, or FTL Engine - Eco stays in your inventory, it does not solve the problem.

<h2 id="permits-knowledge-analysis">
  Permits, Knowledge, and Laboratory analysis
</h2>

Permits are not just side upgrades. They decide when you can move from FTL to cargo, ship parts, Personal Factory, Automated Factories, Modular Systems 2, Plumbing, Liquid Logistics, and later progression routes.

The safe rule is: **buy the permit that solves your current blocker**.

<ul className="my-6 list-disc space-y-2 pl-6">
  <li><strong>Electromagnetism</strong> when the FTL route asks for it.</li>
  <li><strong>Cargo Accommodation</strong> when storage becomes the problem.</li>
  <li><strong>Personal Factory</strong> when you are ready to start a base.</li>
  <li><strong>Automated Factories</strong> after the first Extractor-to-Smelter loop works.</li>
  <li><strong>Modular Systems 2</strong> when you need Cargo Hold, Hi-Pi Laser, Module Kit, or Strong Metal Sheet.</li>
</ul>

Some missing recipes are not permit problems. They may require **Laboratory analysis**, **Science level**, **Technology XP**, or **Exploration progression** instead.

For the full permit table, Modular Systems 2, Lab analysis, Science XP, Technology XP, and progression blockers, read the <Link href="/spacecraft/permits-knowledge-unlock">SpaceCraft Permits and Knowledge Unlock Guide</Link>.

<h2 id="base-building-automation">
  Base building and automation
</h2>

Base building is where SpaceCraft starts becoming a production game, but the first goal is not a giant factory.

Start with one simple loop:

**unclaimed deposit → Base Command Center → Extractor → Smelter → Pylon → Drone Route**

That single loop teaches the important rules: deposits matter, power must be connected, Smelter input must be selected, and Drone Routes require an actual Drone before the route can work.

<h2 id="faq">
  SpaceCraft FAQ
</h2>

### What is this SpaceCraft guide for?

This SpaceCraft guide is a first-10-hours route hub. It helps you move from the beginner mining route into Scanalyzer Alpha, FTL Engine - Eco, resources, ship upgrades, Knowledge permits, Noviciate, corporation access, and first base automation.

### What should I do first in SpaceCraft?

Start with the Beginner Guide route: mine Copper and Iron on Solar Alpha 1, return to Babylon, craft and install Scanalyzer Alpha, then go to Solar Alpha 2 for Gray Quartz and Gravite.

### I do not know what to do next. Where should I start?

Use the “last thing you finished” table near the top of this hub. It points you to the next guide based on your current progress.

### How do I unlock FTL?

Use the Scanalyzer Alpha and FTL Guide. The short path is Modular Systems, Scanalyzer Alpha, Gray Quartz, Crystalline Focuser, Gravite, Silicon Gel, Electromagnetism, FTL Engine - Eco, and Magnetic Plasma.

### Why is Scanalyzer Alpha not working?

It is probably crafted but not installed. Install it in the Shipyard and confirm the edit before returning to Solar Alpha 2.

### Where do I find Gray Quartz?

Gray Quartz appears on Solar Alpha 2 after Scanalyzer Alpha is installed. Scan it first, then mine it.

### How do I refill Magnetic Plasma?

Use the station ship or dock prompt before the first FTL jump. Installing FTL Engine - Eco is not enough if Magnetic Plasma is empty.

### Where do I find Carbon, Silicate, or Pyrite?

Use the Mining Bureau after RakMu. Bureau targets and paid coordinates are the cleanest way to find B-Carbon, Silicate, and Pyrite.

### What permit should I buy first?

Buy the permit that solves your current blocker. Electromagnetism helps the FTL route, Cargo Accommodation helps storage, Personal Factory starts base building, and Automated Factories supports drones and storage after your first loop works.

### Why do I have materials but no recipe?

Check Knowledge permits, Laboratory analysis, blueprint access, and Science / Technology / Exploration progression. Some recipes need more than materials.

### What ship upgrade should I get first?

Cargo Hold is usually the best first upgrade because the starter ship fills quickly and limits every resource run. If searching is the problem, prioritize Resource Detector instead.

### What is System Support?

System Support is the ship’s support limit for installed modules. If module SP requirements get too high, heat becomes harder to manage.

### How do I unlock corporation access?

Reach RakMu, talk to Ingrid, pass the Noviciate cognitive and physical assessment, then use the corpo interface to create or join a corporation.

### When should I build a base?

Start base building after Personal Factory and the base route unlocks, but do not rush a huge layout. Your first goal is one Extractor, one Smelter, one Pylon connection, and one Drone Route.

### How do I find an unclaimed base deposit?

Use the Base Building and Automation Guide. The short version is to scan, check the map, fly low, confirm the deposit is not already claimed, then place the Base Command Center.

### Is this a complete endgame guide?

No. This hub focuses on the opening route, early FTL, resources, permits, Noviciate, ship upgrades, and first automation. Endgame logistics should be expanded after more stable route data is available.

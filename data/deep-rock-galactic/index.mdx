<h2 id="rogue-core-guide-overview">
  Deep Rock Galactic: Rogue Core Guide Hub
</h2>

Deep Rock Galactic: Rogue Core is not just another Deep Rock Galactic mission type.

It is a run-based roguelite spin-off where your build develops during the dive. You choose a Reclaimer, enter a facility, collect Expenite, trigger upgrades, adapt your weapons and abilities, and try to survive long enough for the run to reach its end.

Current Early Access coverage to remember:

- **1-4 player co-op** with solo support.
- **Five Reclaimers:** Guardian, Falconer, Spotter, Slicer, and Retcon.
- **100+ upgrade pool** across changing Early Access systems.
- **Standalone game**, not DLC for the original Deep Rock Galactic.
- **Run-based progression**, not classic DRG mission pacing.

This page is the entry point for the Rogue Core guide cluster. It is not meant to replace the deeper guides below.

<h2 id="quick-game-definition">
  Rogue Core in One Minute
</h2>

Rogue Core is a co-op roguelite FPS where each run is built around adaptation.

You do not start with a finished DRG-style kit. You start with a Reclaimer, basic gear, and a mission route. During the run, you add weapons, class upgrades, Expenite upgrades, Bio Boosters, Workbench choices, and side rewards.

That is the core difference:

**Deep Rock Galactic is mission-first. Rogue Core is run-first.**

<h2 id="faq">
  FAQ
</h2>

<h3 id="what-is-rogue-core">
  What is Deep Rock Galactic: Rogue Core?
</h3>

Deep Rock Galactic: Rogue Core is a standalone 1-4 player co-op FPS action roguelite spin-off of Deep Rock Galactic. It uses Reclaimers, Expenite upgrades, run-based dives, and randomized build decisions instead of classic DRG mission pacing.

<h3 id="is-rogue-core-dlc">
  Is Rogue Core DLC for Deep Rock Galactic?
</h3>

No. Rogue Core is a standalone game, not DLC. You do not need to own the original Deep Rock Galactic to play Rogue Core.

<h3 id="which-guide-should-i-read-first">
  Which Rogue Core guide should I read first?
</h3>

If you are new, start with the <Link href="/deep-rock-galactic/beginner-guide">Beginner Guide</Link>. If you already know the basics and need to choose a class, read the <Link href="/deep-rock-galactic/best-class">Best Class Guide</Link>. If your runs fall apart after a few upgrades, read <Link href="/deep-rock-galactic/upgrades-explained">Upgrades Explained</Link>.

<h3 id="is-rogue-core-good-for-solo">
  Is Rogue Core good for solo?
</h3>

Yes, but solo has its own pressure. Cooper helps, but every interaction still falls on you. Use the <Link href="/deep-rock-galactic/solo-guide">Solo Guide</Link> and <Link href="/deep-rock-galactic/cooper-guide">Cooper Guide</Link> if you mostly play alone.

<h3 id="what-is-the-best-class">
  What is the best class in Rogue Core?
</h3>

There is no single best class for every player. Guardian is the safest starter, Falconer is flexible in co-op, Spotter is team support, Slicer is aggressive solo pressure, and Retcon is the high-skill timing pick.

<h3 id="what-is-expenite">
  What is Expenite in Rogue Core?
</h3>

Expenite is the main run upgrade resource. You mine it, deposit it into the R.E.P.D., fill the upgrade bar, and choose upgrades that shape your current run.

<h3 id="should-i-read-a-wiki-first">
  Should I read a wiki before playing Rogue Core?
</h3>

Not for your first runs. A wiki is useful for exact data, but early on you need decision help: what to do first, which class to try, how to use Expenite, and when to leave a floor.

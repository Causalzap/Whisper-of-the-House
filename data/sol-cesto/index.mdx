<h2 id="start-here-if-you-are-new">
  Start here if you are new
</h2>

If you are new to Sol Cesto, the easiest mistake is assuming every unlock works the same way.

They do not.

The cleanest way to understand the game’s guide topics is:

- some characters unlock through **normal progression**
- some unlock through **clear-based conditions**
- some systems are **hidden mechanics**, not character routes

That is why these four pages matter most right now:

- How to Unlock Lizard
- How to Unlock Huntress
- Flute Guide
- All Characters Unlock Guide

If you only want one entry point, start with the <Link href="/sol-cesto/all-characters-unlock-guide">All Characters Unlock Guide</Link>.

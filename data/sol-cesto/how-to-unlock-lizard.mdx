<h2 id="how-to-get-the-egg">
  How to get the Egg in Sol Cesto
</h2>

The Lizard unlock starts with the **Egg**.

You get it from the **shop**, not from a permanent unlock tree and not from a hidden biome event.

That matters because many players get stuck before the real unlock attempt even starts.

### Best way to find the Egg

The most reliable approach is:

1. reach a shop as early as possible
2. check whether the Egg is being sold
3. if it is not there, use **dice rerolls**
4. buy the Egg immediately once it appears

Buying it early gives you a much better chance of actually completing the hatch condition.

If you wait too long, the problem is not usually “wrong method.”

The problem is just that the run is already too unstable.

### Step 5: Return to the main menu after the run

Once the Egg has cracked, the unlock is effectively complete.

<h2 id="do-you-need-the-candle">
  Do you need the Candle?
</h2>

No.

This is the biggest piece of misinformation around the Lizard unlock.

Early player theories suggested that you needed to:

- buy the Candle
- keep it near the Egg
- use heat to hatch it

That version is no longer the best-supported explanation.

The stronger community consensus is:

- **the Candle is not required**
- **Biome 3 is the actual trigger**
- the “heat” idea is better understood as a **depth / world-logic explanation**, not an item requirement

<GuideCallout title="Candle myth, explained">
If you followed an older guide that told you to buy both the Egg and the Candle, that guide was working from an early community theory.

The cleaner current answer is:

**Egg + Biome 3 = hatch.**
</GuideCallout>

<h2 id="why-your-egg-is-not-hatching">
  Why your Egg is not hatching
</h2>

If your Egg is not hatching, one of these is usually the reason:

- you did **not reach Biome 3**
- you died too early
- you bought the Egg too late in the run
- you no longer had the Egg in your inventory
- you followed outdated Candle instructions instead of just pushing deeper

This is why early purchase matters so much.

The unlock is easy to explain, but not always easy to execute if the run is already falling apart.

### The fastest mental check

Ask these three questions:

1. **Did I buy the Egg?**
2. **Was the Egg still in my inventory?**
3. **Did I actually reach Biome 3?**

If the answer to any of those is no, the unlock attempt was not complete.

<h2 id="what-if-the-run-fails">
  What if the run fails before the Egg hatches?
</h2>

You can retry.

If the run ends before the Egg cracks, just start a new run and repeat the process:

1. find the shop
2. reroll if needed
3. buy the Egg again
4. reach Biome 3

There is no permanent lockout from failing one attempt.

You do **not** ruin the unlock by losing a run.

You only need one successful attempt where the Egg actually cracks.

<GuideCallout title="Retry rule">
If the Egg never cracked, the attempt did not count.

Just buy another Egg on the next run and try again.
</GuideCallout>

<h2 id="faq">
  FAQ
</h2>

### Where do you find the Egg in Sol Cesto?

You find the Egg in the **shop**. If it does not appear, use **dice rerolls** until it does.

### Do you need the Candle to unlock Lizard?

No. The Candle is not required.

### When does the Egg hatch?

The Egg hatches when you reach **Biome 3**, as long as it is still in your inventory.

### Can the Egg hatch in Biome 2?

There is no strong current evidence that Biome 2 is the reliable trigger. Biome 3 is the safest answer.

### What if my run fails before the Egg cracks?

Just start another run, buy the Egg again, and try again. A failed attempt does not permanently block the unlock.

### What if the Egg already cracked and then I die?

That should still count, because the important part is that the hatch already happened before the run ended.

### What does Lizard do?

The unlocked character panel shows that Lizard has **7 HP, 2 Strength, 1 Magic**, and moves randomly into one of the four corners.

### Is Lizard the secret character?

Lizard is the hidden Egg-based character unlock that most players are referring to when they talk about the game’s secret character.

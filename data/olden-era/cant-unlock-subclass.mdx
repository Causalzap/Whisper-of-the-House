<h2 id="wrong-base-class">
  Wrong base class route
</h2>

<GuideTable
  caption="Same faction, different base class, different subclass route."
  columns={baseClassColumns}
  rows={baseClassRows}
/>

A common mistake is checking only the faction.

That is not enough. Advanced Class formulas are tied to the hero's **base class**. A Necropolis hero is not automatically using every Necropolis subclass formula.

To find your base class in-game, open the **hero information panel** and check the class line near the hero name and starting skills. That class label is the route filter you should use before following any Advanced Class formula.

Check the base class before checking the five skills. If the route belongs to another class, the subclass will not unlock no matter how many similar skills you take.

<h2 id="missing-required-skill">
  Missing required skill
</h2>

A required skill cannot be replaced by a similar skill.

This is where many players get trapped. They pick a skill that sounds right for the class, but the formula asks for something else.

Examples:

- a Death Knight route may ask for Intelligence or Diplomacy
- a Necromancer route may ask for Economy or Logistics
- a Temple route may ask for a magic school that does not match your current spell plan
- a route may need Scouting even if you wanted a direct combat skill

</h2>

<GuideTable

The route is recoverable when the problem is only skill level.

<h2 id="early-access-data">
  Early Access data changed
</h2>

Olden Era entered Early Access on April 30, 2026, so subclass data should be treated as patch-sensitive.

Recheck after major updates if:

- the formula no longer unlocks
- a required skill stopped appearing
- a passive behaves differently
- UI route markers changed
- class skill pools changed
- patch notes mention skills or Advanced Classes

<GuideCallout title="Early Access rule">
If a route worked before and no longer works, check patch data before rewriting your whole build plan.
</GuideCallout>

<h2 id="what-to-do-instead">
  What to do if the subclass route is dead
</h2>

If the route is dead, stop spending levels trying to save it.

A strong fallback plan is:

- take the best combat skill upgrades available
- improve map tempo with movement or economy
- focus on army preservation
- build around your current skill set
- stop chasing the passive
- use the next hero or next map for a cleaner route

<GuideCallout title="Pivot rule">
A strong normal hero is better than a failed Advanced Class project. Once the formula is impossible, every extra level spent forcing it is a tempo loss.
</GuideCallout>

<h2 id="faq">
  FAQ
</h2>

### Why can’t I unlock my subclass?

Usually one required skill is missing, one required skill is not Expert, the formula belongs to the wrong base class, your skill slots are full, or the rare bottleneck skill never appeared.

### Do Basic or Advanced skills count?

No. Only Expert counts as a completed Advanced Class requirement.

### Can I unlock a subclass with four required skills?

No. You need all five required secondary skills, and all five must be Expert.

### Where do I check my hero's base class?

Open the hero information panel and look near the hero name, starting skills, or class details. The base class decides which Advanced Class formulas can unlock.

### Can the same faction have different subclass routes?

Yes. Subclass routes depend on base class, not only faction. For example, Necromancer and Death Knight have different Advanced Classes.

### What if the missing skill never appears?

Stop forcing if the missing skill is a rare bottleneck and the route is hurting your tempo.

### Can I recover a failed subclass route?

You can recover if the only problem is upgrading a required skill to Expert. You usually cannot recover if the base class is wrong, slots are full, or the bottleneck skill never appeared.

### Is the subclass system bugged?

Usually no. Most failed unlocks are caused by missing requirements or non-Expert skills. But because Olden Era is in Early Access, patch changes can affect formulas and skill behavior.

### Should I keep leveling until the subclass unlocks?

Only if the route is still mathematically possible and the passive will matter in time. Otherwise, pivot to a strong normal build.

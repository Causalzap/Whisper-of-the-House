<GuideQuickAnswer title="Quick Answer">
If **Try for Baby is not showing in Paralives**, check the pregnancy settings before you blame the bed. One Para needs **Can Get Pregnant**, and the other needs **Can Get Others Pregnant**. If those settings do not match, the relationship can be ready and the bed option can still fail.

After that, check the route: become romantic interests and lovers, use a baby-related Together card such as **Ask if they want to try for a baby NOW** or **Discuss having children in the future**, select both Paras, then click a **bed**. If the action finishes but stays stuck in the queue, cancel the finished action manually before you reset the whole setup.

<h2 id="fertility-settings">
  Check pregnancy ability settings first
</h2>

This is the first thing I would check when Try for Baby is missing.

In Paramaker, open the advanced personality or gender-related options for both Paras. The biological route needs one Para who can get pregnant and one Para who can get others pregnant.

  caption="Pregnancy ability settings needed for Try for Baby."

If both Paras are set the same way, the relationship route can look correct but the bed option may still be missing. That is why I check this before I move furniture, replace the bed, or retry the interaction again and again.

If the household already exists, open the cheat panel with **Ctrl + F1** or **Ctrl + Shift + C**, type `editcharacter`, and check the relevant Para in Paramaker. Save before editing if this is a main household.

<h2 id="pregnancy-test-negative">
  Pregnancy test is negative
</h2>

A negative pregnancy test is not proof that the save is broken. If the test is negative, I check whether the interaction completed cleanly, then retry once after confirming the settings and bed route.

If you get several negative results, check pregnancy chance settings before assuming the bed is bugged. Some saves may have lower odds than you expect.

<h2 id="adoption-and-paramaker">
  Adoption and Paramaker are separate family routes
</h2>

Try for Baby is not the only way to add a child to a household.

If the biological route is not what you want, use adoption instead. The adoption route belongs in a dedicated guide because it has a different flow, usually involving public services such as the hospital or pharmacy rather than bed interactions.

You can also create a child directly in Paramaker if you are building a household setup from scratch. That is useful when you care about the family layout more than playing through pregnancy.

I keep those routes separate in my head. Try for Baby is a relationship and pregnancy route. Adoption is a household expansion route. Paramaker child creation is a setup route.

<h2 id="faq">
  FAQ
</h2>

### Why is Try for Baby not showing in Paralives?

Check pregnancy ability settings first. One Para needs Can Get Pregnant and the other needs Can Get Others Pregnant. Then check romance, lover status, the baby-related Together card, both Paras selected, and the bed.

### How do I change pregnancy ability settings?

Open the cheat panel with **Ctrl + F1** or **Ctrl + Shift + C**, type `editcharacter`, and check the Para's advanced personality or gender-related options in Paramaker.

### Which Together card do I need?

Look for baby-related cards such as **Ask if they want to try for a baby NOW** or **Discuss having children in the future**.

### Do I need to select both Paras?

Yes. Select both Paras before clicking the bed.

### Can I Try for Baby in the shower?

I would not use the shower for the baby route. Use a bed.

### What should I do if Try for Baby gets stuck?

If the animation already finished, cancel the stuck action manually. If it still fails, check bed space and retry from a clean save.

### Why is the pregnancy test negative?

A negative test can simply mean the attempt failed. Check the setup, pregnancy chance, and whether the interaction completed cleanly.

### Can I adopt instead?

Yes. Adoption is a separate route and is better treated as its own guide because the steps are different from Try for Baby.

### Can I make a child in Paramaker?

Yes, use Paramaker when you want to build a household directly instead of playing through pregnancy.

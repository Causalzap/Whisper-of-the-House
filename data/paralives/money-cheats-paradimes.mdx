<GuideQuickAnswer title="Quick Answer">
To use **Paralives money cheats**, open the **Cheat Commands** panel with **Ctrl + F1** or **Ctrl + Shift + C** on Windows. On Mac, try **Command + Fn/Globe + F1**. Then type a command and press Enter.

Use `payday` for the selected Para's salary, `piggybank` for **₱1,000**, `makeitrain` for **₱10,000**, `jackpot` for **₱50,000**, `printmoney 5000` to add an exact amount, `setmoney 100000` to set household funds to an exact total, `lottery` for a random cash result, or `clearbills` when bills are the only problem.

Before using big money changes, I make a manual save. Money cheats may be listed as achievement-safe by cheat indexes, but a clean reload point still matters if the new budget makes the save less fun.

<h2 id="best-money-cheat">
  Which money cheat should you use?
</h2>

Do not use every money command the same way. The best command depends on whether you want to keep a normal first save, recover from a small mistake, build freely, or reset the household to an exact budget.

<GuideTable
  caption="Best Paralives money cheat by situation."
  columns={decisionColumns}
  rows={decisionRows}
/>

My rule is simple: use the smallest cheat that solves the problem. If I only need to cover a bill or buy one object, I use `payday` or `piggybank`. If I am testing a house layout or placing an expensive Workshop lot, I use a bigger or more exact command.

<h2 id="open-cheat-panel">
  How to open the Cheat Commands panel
</h2>

On Windows, press **Ctrl + F1**. If that does not work on your keyboard, try **Ctrl + Shift + C**. On Mac, try **Command + Fn/Globe + F1**.

The panel appears on the right side of the screen with a text box at the bottom. Click inside the box, type the command, and press **Enter**. If the command works, the panel confirms the transfer with a success message.

Commands are not case-sensitive. I still type them in lowercase because it is easier to read and harder to mistype.

<h2 id="save-before-cheats">
  Save before large money changes
</h2>

Small money cheats are usually harmless, but I still save before using big commands like `jackpot`, `lottery`, `printmoney`, or `setmoney`.

The reason is not that money cheats are complicated. The problem is that they can change the feel of a save quickly. A household that was fun at ₱6,000 may feel too easy at ₱100,000. A downloaded house may also push you into a different style of play than you planned.

I make a manual save, test the money change, then continue only if the new budget still fits the story I want to play.

<h2 id="printmoney-vs-setmoney">
  Printmoney vs setmoney: add money or set the total
</h2>

This is the money cheat difference that matters most.

  caption="Printmoney vs setmoney in Paralives."

Use `printmoney` when you like your current balance but need more. Use `setmoney` when you want the final household funds to be a specific number.

For example, `printmoney 5000` is better when you are short by about ₱5,000. `setmoney 100000` is better when you are starting a build test and want the household to begin with exactly ₱100,000.

<h2 id="lottery">
  Lottery: random money
</h2>

Use `lottery` when you want money randomness to be part of the save.

This command is less practical than `printmoney` or `setmoney` because you are not choosing the amount. That makes it fun for casual play, but weak for exact building plans.

I would not use `lottery` when I need to afford a specific Workshop house or finish a specific room. I would use it when I want the save to feel like a lucky event happened and I am willing to accept whatever number I get.

<h2 id="clearbills">
  Clearbills: fix bills without changing the whole budget
</h2>

Use `clearbills` when bills are the problem.

This is different from adding money. If your household balance is fine but unpaid bills are blocking the save, clearing bills can be cleaner than using a huge money cheat.

I like this command because it solves a specific problem. Instead of turning a struggling household into a rich household, it removes the bill pressure and lets the normal routine continue.

<h2 id="best-cheat-for-building">
  Best money cheat for building
</h2>

For building, I choose based on how controlled I want the budget to be.

Use `printmoney` when you know exactly how much more you need. Use `setmoney` when you want a controlled build budget from the start. Use `jackpot` when you want to get moving quickly and do not care about exact funds. Use `makeitrain` when the goal is only one room or one upgrade pass.

This matters even more if you are using downloaded lots. A Workshop home can cost more than your starter household can afford, so small commands like `payday` may not solve the problem. For lot placement and modded homes, see <Link href="/paralives/best-mods-steam-workshop-install-guide">Paralives Mods Guide</Link>.

<h2 id="best-cheat-for-normal-play">
  Best money cheat without ruining progression
</h2>

For a normal first save, I would avoid `jackpot`, `lottery`, and large `setmoney` values unless I am deliberately testing.

Use `payday` when you want the smallest correction. Use `piggybank` when you need a safe early cushion. Use `makeitrain` only when the house layout is blocking basic play.

Money cheats are most useful when they remove a bad bottleneck. They are less useful when they erase every decision. If your goal is to learn careers, skills, needs, and daily routine, keep the cheat small.

If you want to see the full command list, type `help` in the Cheat Commands panel.

<h2 id="faq">
  FAQ
</h2>

### How do I open money cheats in Paralives?

On Windows, use **Ctrl + F1** or **Ctrl + Shift + C**. On Mac, try **Command + Fn/Globe + F1**.

### Do I need to enable cheats first?

No. Open the Cheat Commands panel, type the command, and press Enter.

### Are Paralives cheat commands case-sensitive?

No. Commands work in upper or lower case. I still type lowercase because it is easier to read.

### Are Paralives money cheats achievement-safe?

Paralives.io lists Paralives cheats as achievement-safe. I still recommend making a manual save before large money changes because a cheat can change the feel of a household even if achievements remain available.

### What is the best Paralives money cheat?

For small fixes, use `payday` or `piggybank`. For exact extra money, use `printmoney`. For exact final funds, use `setmoney`. For building quickly, use `jackpot`.

### How much money does payday give?

`payday` gives the selected Para's salary. In my test save, it added **+₱200**.

### What is the difference between printmoney and setmoney?

`printmoney` adds the amount you type. `setmoney` changes the household balance to the amount you type.

### What does lottery do?

`lottery` gives a random money result, so I only use it when I do not need an exact amount.

### How do I see all cheat commands?

Type `help` in the Cheat Commands panel.

### Should I save before using money cheats?

Yes. I make a manual save before large money changes so I can reload if the new budget makes the save less fun.

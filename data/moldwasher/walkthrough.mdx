<h2 id="opening-and-first-fridge-jobs">
  Opening and first fridge jobs
</h2>

At the start, clean the door, enter the house, clear the small home objects, then open the map. I do not skip the home shelf area because early pickups teach how later collectibles work.

In the first fridge jobs, I use wide spray for open mold and precise stream for stubborn small patches. The main mistake is leaving one tiny living patch behind and letting it spread again.

<h2 id="sticker-fridge">
  Sticker fridge: unauthorized stickers and precise stream
</h2>

The sticker fridge is the best early lesson for precise spray. The floor dirt is not the important part at first; the sticker layer is.

My order is:

1. Start with the sticker cluster.
2. Use the precise stream instead of wide spray.
3. Hold the stream on sticker edges until the sticker layer clears.
4. Check the covered area before leaving.
5. Switch back to wide spray for normal mold and floor cleanup.

This is also a useful hidden-drop habit: when the game asks me to remove a cover layer, I always check what was hidden behind it before pressing Space.

<h2 id="fly-fridge">
  Fly fridge mission: dirty tracks and eggs
</h2>

The mission starts with a fly leaving dirty tracks while you are trying to clean the fridge.

My order is:

1. Spray the fly until it drops or stays controlled long enough to work.
2. Clean the eggs and nearby floor while the fly is not moving around.
3. Sweep the main fridge floor again because the fly may have re-dirtied it.
4. Check the clear screen before leaving.

<h2 id="dog-bowl">
  Dog bowl mission
</h2>

The prompt is about cleaning the bowl before the dog returns, but the real trick is positioning.

My order is:

1. Clear the easy dirt around the mat first.
2. Use the can / bowl shape as a ramp.
3. Get into the upper or inside bowl angle.
4. Clean the hard-to-reach surface from that position.
5. Finish the remaining floor patches last.

<h2 id="sink-clogged-route">
  Sink / clogged route: stuck at 0% or not progressing
</h2>

The sink route can feel confusing because the first step is not normal washing.

My order is:

1. Start with the clog or blocked part of the sink.
2. Use the required tool or interactable object to open the route.
3. Clean the sponge, drain, and inner basin.
4. Sweep the rim and edge objects once more before leaving.
5. Check whether a reward or collectible appears after the drain is handled.

The reason it can look stuck at 0% is that visible dirt is not the real first objective. Once the clog is fixed, the rest behaves like a normal sink-cleaning route.

<h2 id="breakfast-station-hidden-drop">
  Breakfast Station / Level 14 hidden drop note
</h2>

Some players search for **Breakfast Station level 14 hidden drop** specifically. I treat it like a hidden-drop diagnosis rather than a normal story blocker.

My order is:

1. Clear the main breakfast mess first.
2. Do not press Space immediately if the clear banner appears.
3. Check optional breakfast-side objects that react to cleaning.
4. Sweep behind props and under object edges.
5. Confirm the result screen. If it still shows **0/1 Hidden Drops**, replay that exact level instead of searching the whole map.

This section is intentionally cautious: I would rather verify the missing drop through the result screen than publish an exact position that may be wrong after a launch patch.

---

<h2 id="pickaxe-hard-objects">
  Pickaxe jobs: cookies, crackers, walls, and hard chunks
</h2>

When the game introduces hard objects, I stop treating them like mold. The pickaxe / axe route is about breaking first, cleaning second.

My order is:

1. Hit the hard object until it breaks apart.
2. Switch back to the washer for dust, grime, or exposed mold.
3. Use the blower only if pieces need to move.
4. Check corners after the object breaks, because new dirt can be exposed behind it.

If water does nothing, that is usually the game telling me to break the object first.

---

<h2 id="rice-table-rubbish-bag">
  Rice table, rubbish bag, and trash cleanup
</h2>

Rice and rubbish bags are object-movement checks. Spraying harder does not solve the real objective.

My order is:

1. Use the leaf blower, not the washer.
2. Push rice toward the bag or target area.
3. If you are moving a rubbish bag, push it toward the disposal / target zone instead of trying to clean it.
4. Work from the outside inward so rice does not scatter behind you.
5. Clean leftover dirt only after the movement objective is handled.

<h2 id="air-duct">
  Air duct and wind jobs
</h2>

The air duct route is not about raw cleaning power. It is about wind direction, object movement, and breaking or positioning rice balls.

My order is:

1. Watch which direction the air is pushing.
2. Break or loosen the rice ball / object if needed.
3. Move the object close enough to the edge or airflow path.
4. Let the wind push it instead of fighting the airflow.
5. Clean the remaining grime after the object movement is solved.

If the object keeps coming back to the middle, change your angle. The goal is not to overpower the wind; it is to make the wind do part of the work.

<h2 id="beef-monster-dinner-fight">
  “This dinner doesn't want to be cooked” walkthrough
</h2>

This is the beef monster / dinner fight. It starts like a normal cleaning job, then becomes a tool-cycle level.

My order is:

1. Start with the washer to reveal the active parts.
2. When water stops helping, switch to the flamethrower.
3. Burn the parts that react to fire.
4. Use the axe / pickaxe on arms, chunks, or exposed pieces that need to be hit.
5. Use the leaf blower when the level asks you to move eyes, pieces, or loose parts.
6. Return to the washer for final residue.

<h2 id="faq">
  FAQ
</h2>

### How do I progress in Moldwasher?

Follow the route table first, then use the stuck points table to jump to the exact mission that is blocking you.

### What is the difference between left click and right click?

Left click is wide spray; right click is precise stream. Use the core mechanics section above for the full control logic.

### What are the mold types in Moldwasher?

Moldwasher includes several mold variants such as colorful, iridescent, ghostly, inverted, sticky, glowing, crystalline, and mimic-style mold. Use the mold types table above for responses.

### Does pausing stop mold growth?

I would not rely on pausing as your safety plan during active mold growth. Use the core mechanics section above and consider Zen Mode if the normal route feels stressful.

### How do I turn on Zen Mode?

Use Zen Mode when you want slower mold pressure and a calmer route. It is best treated as a comfort option rather than a separate walkthrough route.

### Where do I get the pickaxe?

In my run, the pickaxe became available after the dog bowl route. See the tool unlock section above before starting pickaxe-gated missions.

### How do I beat the sink level if it is stuck at 0%?

Use the sink / clogged route section above. The short version: the clog is the real first blocker.

### How do I clean rice or dispose of rubbish bags?

Use the rice table and rubbish bag section above. The short version: use the blower and move the object into the target area.

### How do I beat the air duct level?

Use the air duct section above. The key is to position the object so the wind helps move it.

### How do I beat “This dinner doesn't want to be cooked”?

Use the dinner fight section above. It is a tool-cycle level, not a single-tool cleaning job.

### What should I do if a level seems bugged?

Switch tools first. If the active object still does not respond to washer, fire, pickaxe / axe, or blower, restart the level and check for updates.

### What should I do after finishing the story?

Move to the dedicated 100% pages: the collectibles guide for hidden drops and Gamer Brain®, and the achievements guide for Sushi Set, HighGrow, Yakimeshi, and all 27 achievements.

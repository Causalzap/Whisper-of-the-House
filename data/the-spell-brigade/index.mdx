<h2 id="faq">
  FAQ
</h2>

### Where should a new player start?

Start with the <Link href="/the-spell-brigade/solo-guide">Solo Guide</Link>, then read the <Link href="/the-spell-brigade/best-builds">Best Builds Guide</Link>. Surviving consistently matters more than optimizing early.

### Which guide should I prioritize after Solo and Builds?

If you want stronger runs, read the <Link href="/the-spell-brigade/best-spells-elements">Best Spells & Elements Guide</Link>. If progression feels slow, read the <Link href="/the-spell-brigade/gold-farm">Gold Farm Guide</Link>. If you are choosing who to play next, read the <Link href="/the-spell-brigade/wizard-unlocks">Wizard Unlocks Guide</Link>.

### Which guide should achievement hunters read?

Start with the <Link href="/the-spell-brigade/achievements-guide">Achievements Guide</Link>, then use the <Link href="/the-spell-brigade/wizard-unlocks">Wizard Unlocks Guide</Link> for wizard-specific routes.

### Are these guides for The Spell Brigade 1.0?

Yes. This guide cluster is written around the 1.0 version structure, including builds, infusions, wizard unlocks, achievements, Trial Covenants, Masteries, and Ascension.

### Which guide should I read for late-game progression?

Read the <Link href="/the-spell-brigade/trial-covenants-masteries-prestige">Covenants, Masteries & Ascension Guide</Link> if you are working on Trial Covenants, Devotion, Banish, rerolls, Masteries, Enchantments, or long-term wizard progression.

DragonSword Awakening gives me several ways to improve one character at once: gear
level, set bonuses, selected main stats, reforged substats, Basic Karma, Exclusive
Karma, and skills. The expensive mistake is treating every system as an equal
priority.

I finish one functional damage package first. That means a useful set structure, a
main stat that supports the real rotation, a matching Basic Karma, and enough skill
investment to clear the next activity. Reforging and Exclusive Karma come later,
after I know the item and hero will remain in the core team.

<h2 id="gear-set-tiers">
  Read gear set tiers before crafting five matching pieces
</h2>

The Azure Gale Top is the clearest complete set example in the current build data.

Its panel shows:

- **2 Set:** Airborne Additional Damage **+15%**
- **3 Set:** Attack Power **+10% for 8 seconds** after using an Active Skill

The same screen shows **5,000 Gold** for the chest craft and three material
requirements with counts of **2, 3, and 3**.

When the displayed bonuses stop at three pieces, I use a **3+2 setup**:

- three pieces from the main set to activate its final listed tier;
- two pieces from another set with a bonus the character or team can trigger.

Five matching pieces can still have individually useful stats, but they do not create
an extra set tier when the panel lists no four-piece or five-piece effect.

The second set follows the actual combat loop. An Airborne bonus is valuable when the
character repeatedly launches or fights in the air. A Stun or Poison bonus is poor
when the current lineup rarely creates that state.

### When I craft and when I wait

I craft when the new item completes a useful set tier, supports the core hero's real
condition, and will survive the next part of progression.

I wait when:

- the next story section is replacing gear quickly;
- the materials only produce one isolated piece;
- the set effect does not match the current team;
- or the recipe is new but does not solve the current damage or survival problem.

One higher-rarity item can still be a downgrade when equipping it breaks a working
set effect.

<h2 id="main-stats-forging-molds">
  Main stats and how Forging Molds work
</h2>

A Forging Mold is used while crafting equipment to select the intended main stat. Its
tooltip states that it is destroyed after use, so I treat every Mold as a one-use
decision and reserve it for a core piece rather than temporary story gear.

Regional Mission rewards are one source I check for Forging Molds.

### Apply the selected main stat before confirming

On controller, opening the stat list is not the same as applying the selection.

My sequence is:

1. open the Blacksmith and highlight the equipment;
2. use the displayed selection input to open the main-stat menu;
3. highlight the intended stat;
4. press the selection input again;
5. verify that the stat remains visibly selected;
6. confirm the craft only after that state is shown.

already good enough to clear the next activity.

<h2 id="basic-karma">

</h2>

I equip a Basic Karma that supports the hero's repeated condition or broad damage
plan.

Examples from the tested builds are:

- Poison support for Charlotte;
- Airborne support for Reina;
- general Crit or offensive support when the build has not settled on one reliable

World Bosses can reward Basic Karma. I inspect the displayed reward pool and choose
the boss that can drop the condition I need, but the actual result remains random.

<h2 id="exclusive-karma">
  Exclusive Karma, Hero Logs, Tower exchange, and dismantling
</h2>

Exclusive Karma can add a character-specific mechanic rather than only a general
stat bonus. Charlotte's option, for example, enables the clone-detonation interaction
used in her build.

The routes shown in the source material are:

- complete an available Hero Log that directly rewards that hero's Exclusive Karma;
- earn exchange currency through Tower of Trials;
- dismantle an unwanted Exclusive Karma and use the returned value toward another
  character.

I spend the exchange currency after confirming:

1. the hero remains in the first three-character team;
2. the exclusive effect changes a skill or mechanic used in combat;
3. the hero already has functional gear;
4. and the team direction is unlikely to change immediately.

### Dismantle only after checking the preview

An unused Exclusive Karma can be converted into exchange value, but the original item
is consumed.

Before confirming, I verify that:

- the item is not equipped;
- it is not needed for an immediate ascension;
- and the preview shows the return I expect.

My compact daily order is:

1. Regional Mission reward that fixes the current Gold, Mold, or skill-material
   bottleneck;
2. World Boss whose displayed pool contains a useful Basic Karma;
3. available Hero Log for a core-team character;
4. story or dungeon progression when the current activity is inefficient.

I leave a farming activity when one failed dodge removes most of a hero's health, the
boss survives every full rotation with little progress, or the reward does not solve
the current build problem.

<h2 id="medal-exchange">
  Use Mission Medal Exchange as a gap closer
</h2>

Mission medals can purchase crafting and skill materials. I use the shop to finish a
specific upgrade rather than trying to empty it.

A purchase is worthwhile when it:

- completes the next useful set tier;
- supplies the missing material for a planned main-stat craft;
- reaches an important skill breakpoint;
- or helps the core hero enter the next activity cleanly.

Buying stacks of every material in advance can trap currency in a character or set I
later abandon. The shop works best after I know the exact missing item.

<h2 id="faq">
  DragonSword Awakening gear and Karma FAQ
</h2>

### Is a five-piece set always better than a 3+2 setup?

No. Read the listed tiers. Azure Gale shows effects at two and three pieces, so the
fourth and fifth pieces do not unlock another tier on that panel. A useful second
two-piece bonus can provide more value.

### What does the Azure Gale set do?

Two pieces give **Airborne Additional Damage +15%**. Three pieces give **Attack Power
+10% for 8 seconds** after an Active Skill.

### How much does the Azure Gale Top cost to craft?

The visible example costs **5,000 Gold** and shows three material requirements with
counts of **2, 3, and 3**. Check the current Blacksmith panel before spending in case
a later patch changes the recipe.

### Is a Forging Mold consumed?

Yes. The tooltip used for this guide states that the Forging Mold is destroyed after
use. Reserve it for a main-stat craft expected to stay in the build.

### When is a 50,000 Gold reforge worth it?

When the equipment is a long-term core piece and the missing line materially changes
the build. Stop when the piece already has its main condition plus two useful
offensive lines.

### Should I upgrade Basic Karma or wait for Exclusive Karma?

Upgrade the matching Basic Karma when it belongs to the core hero and the Exclusive
option is not immediately obtainable. Keep the investment lower when it is only a
short-term bridge.

### Can I target a specific Basic Karma from a World Boss?

You can choose a boss whose displayed reward pool contains the condition you want,
but the actual reward is random.

### Should I dismantle an unwanted Exclusive Karma?

Check whether it is equipped or needed for ascension, then inspect the return preview.
Dismantling is useful when the recovered value advances an Exclusive Karma for a hero
you actually use.

### Does the Regional Mission reset use local or server time?

The shown English panel says **“Resets daily at 9:00 AM”** without identifying the
time zone. Use the in-game countdown or observe one reset before setting a fixed
schedule.

<h2 id="burglin-gnomes-beginner-guide">
  Burglin' Gnomes beginner guide
</h2>

**Burglin' Gnomes** looks like a goofy co-op stealing game, but the early runs punish players who ignore the task list. The house is oversized, the old man can trap you, rats and roaches interrupt routes, bees can ruin outdoor tasks, and your own tools can hurt you if you carry them badly.

The beginner rule is simple:

<strong>Reach the crown target, keep the team recoverable, and extract before greed turns into a wipe.</strong>

This guide is for your first clean clears. It focuses on the hard basics: controls, task targets, safe entry, noise, rescue, and when to stop looting.

If you want the full cluster, start from the <Link href="/burglin-gnomes">Burglin' Gnomes Guide Hub</Link>. For objective routes, use the <Link href="/burglin-gnomes/tasks-guide">Tasks Guide</Link> after this page.

<h2 id="minimum-tasks">
  High-Gnome tasks: use the crown target
</h2>

Steam confirms the core rule: the High-Gnome will not carry you around for free, and failing to complete enough tasks can get you fired. The official store page does not list a fixed task number, so do not treat 3 tasks as a universal hardcoded rule.

In the current Gnome Report example used in this guide, the crown target sits around **3/5**. That makes **3 completed tasks** the safest beginner planning target for this version of the run: reach the crown target first, then decide whether extra objectives are worth the risk.

<GuideTable
  caption="Burglin' Gnomes task completion rules for beginners."
  columns={passRuleColumns}
  rows={passRuleRows}
/>

The clean decision is:

<strong>Before the crown target, play for checkmarks. After the crown target, play for extraction.</strong>

If your own Gnome Report, map, or task list shows a different target, follow the in-game UI.

<h2 id="controls">
  Basic controls you should know first
</h2>

Before learning advanced tactics, make sure you know the core controls. Most beginner failures come from not dropping items, not checking tasks, or not using crouch and sprint at the right time.

<GuideTable
  caption="Burglin' Gnomes basic controls."
  columns={controlsColumns}
  rows={controlsRows}
/>

The two controls new players forget most are **Tab** and **Q**.

Use **Tab** before entering deeply so the team knows the task target. Use **Q** when you need to drop a pot, frying pan, weapon, or risky item before escaping or rescuing.

<h2 id="best-starting-map-difficulty">
  Start on the easiest map and difficulty
</h2>

For your first clears, do not start with the largest or most chaotic setup if the game gives you a choice. Pick the smallest / easiest option available, such as **Cottage on Easy**, then learn the route before moving up.

A smaller first setup helps you learn:

<ul className="my-6 list-disc space-y-2 pl-6">
  <li>where windows and low entry points usually are,</li>
  <li>how far you can carry objects before danger builds,</li>
  <li>how the old man moves through rooms,</li>
  <li>which objects are easy task targets,</li>
  <li>and how long it takes your team to reach the crown target.</li>
</ul>

Harder maps are not just bigger. They create more ways for the team to split, panic, and fail rescues.

<h2 id="copy-paste-checklist">
  Copy-paste checklist for your squad

Use this in Discord, Steam chat, or your group message before starting a run.

```txt
BURGLIN' GNOMES FIRST CLEAR PLAN

GOAL:
- Complete enough High-Gnome tasks to avoid getting fired.
- Use the Gnome Report crown marker as the real target.
- In this example, plan around 3 completed tasks.
- After the crown target is reached, stop forcing risky objectives.

- WASD = move
- Shift = sprint
- Ctrl = crouch / quieter movement
- E = interact / pick up / untie
- Q = drop or throw held item
- Tab = task list / journal

ROLES:
- Scout: find rooms, task items, and hazards.
- Hauler: move heavy task items in stages.
- Distractor: pull old man pressure only when needed.
- Anchor: stay near exit, stage loot, rescue teammates.

UPGRADES:
- Movement first.
- If we survive but leave loot behind, buy carry support next.
- If we get caught too early, buy stealth / noise control next.

RULES:
- Start with easy tasks first.
- Use windows instead of forcing tall doors.
- Drop heavy items before rescue.
- Do not start enemy tasks without the right tool.
- When tornado appears, leave.
```

This checklist is intentionally simple. If the team cannot follow this, adding more strategy will not help.

  Use windows instead of forcing tall doors
</h2>

You are a tiny gnome in a human-sized house. That means doors, handles, counters, cabinets, and furniture are part of the puzzle.

A good entry route should also be a good exit route. Before you drag a pot, frying pan, metal can, or other bulky object, ask where it will leave the house.

<h2 id="task-priority">
  Pick tasks by risk, not by excitement
</h2>

The task list is not just a checklist. It is a risk map.

<GuideTable
  caption="Burglin' Gnomes beginner task priority."
  columns={taskPriorityColumns}
  rows={taskPriorityRows}
/>

A strong beginner route usually looks like this:

<ul className="my-6 list-disc space-y-2 pl-6">
  <li>first checkmark from an easy interaction,</li>
  <li>second checkmark from a nearby room or break task,</li>
  <li>third checkmark from a carry task or another safe objective,</li>
  <li>then extraction unless the next task is close and safe.</li>
</ul>

This is why enemy tasks should not usually be first. A rat task can be worth doing, but only after you have the right tool and enough room to miss safely.

<h2 id="noise-system">
  Noise is the hidden timer
</h2>

Noise is not just flavor. It is one of the reasons easy runs turn dangerous. Sprinting, breaking objects, throwing items, opening objects, dragging heavy loot, fighting enemies, and using weapons can all push the run toward detection and pressure.

<GuideTable
  caption="Burglin' Gnomes noise sources and beginner fixes."
  columns={noiseColumns}
  rows={noiseRows}
/>

The practical rule is:

<strong>Spend noise only when it buys a task, a rescue, or an escape.</strong>

Breaking an item because the task asks for it is useful. Breaking random objects while the scout is still looking for the first room is just making the house harder.

<h2 id="weapon-safety">
  Weapon safety: use tools for a job, not for chaos
</h2>

Weapons and dangerous items can save a run, but they can also ruin it. Knives, guns, grenades, bear traps, landmines, and other tools should solve a specific problem.

<strong>Do not climb, rescue, or squeeze through furniture while carrying a dangerous item at a bad angle.</strong>

A safe weapon rule:

<ul className="my-6 list-disc space-y-2 pl-6">
  <li>Use a knife or taser when the task or enemy requires it.</li>
  <li>Do not wave sharp items around teammates.</li>
  <li>Drop dangerous tools before CPR or untie rescues.</li>
  <li>Do not carry explosives as normal loot.</li>
  <li>If the team is panicking, dropping the weapon is usually safer than forcing a fight.</li>
</ul>

This is one of the fastest ways to improve. Many beginner deaths come from the team hurting itself.

<h2 id="upgrade-priority">
  What to upgrade first
</h2>

Do not buy upgrades only because they look funny. Early upgrades should fix the problem that is actually costing your runs.

<GuideTable
  caption="Burglin' Gnomes beginner upgrade priority."
  columns={upgradeColumns}
  rows={upgradeRows}
/>

The safest default is:

<strong>Movement first, then choose carry or stealth based on what keeps ruining your runs.</strong>

If your team usually survives but leaves too much loot behind, carry support can come before stealth. If your team gets detected, grabbed, or trapped too early, stealth and noise control matter more. Special gear is better after basic task clears feel reliable.

For material names and upgrade planning, use the <Link href="/burglin-gnomes/materials-upgrades-guide">Materials and Upgrades Guide</Link>.

<h2 id="tornado-extraction">
  Leave when the tornado appears
</h2>

The tornado warning is your hard stop. Once it appears, the question is no longer “what else can we do?” The question is “what can we safely cash in?”

<figure className="my-8">
  <Image
    src="/images/burglin-gnomes/burglin-gnomes-tornado-extraction.webp"
    alt="Burglin' Gnomes out of time warning telling players to get to the tornado"
    width={1600}
    height={900}
    className="rounded-xl"
  />
  <figcaption className="mt-3 text-sm text-slate-400">
    When the tornado warning appears, stop starting new tasks and get out with the progress already earned.
  </figcaption>
</figure>

Leave immediately if:

<ul className="my-6 list-disc space-y-2 pl-6">
  <li>you already reached the crown target,</li>
  <li>the next objective is in another room,</li>
  <li>the old man is between you and the task,</li>
  <li>a teammate is tied up too far away,</li>
  <li>or the carry item is not already staged near an exit.</li>
</ul>

One missed task is easier to accept than losing the whole squad.

<h2 id="faq">
  Burglin' Gnomes beginner FAQ
</h2>

### How many tasks do I need to complete?

Steam says you must complete enough High-Gnome tasks, but it does not give a fixed number. Use the Gnome Report crown marker as your real target. In the example used here, that target is around 3 completed tasks.

### Can I extract before reaching the crown target?

You can try to leave, but you should not plan around it. If the Gnome Report target is not met, the run may fail even if you reached extraction.

### Should I keep playing after reaching the crown target?

Only if the next task is close, safe, and does not risk a wipe. After the crown target is reached, extraction becomes more important than greed.

### What if my Gnome Report shows a different target?

Follow your current in-game UI. The beginner route on this page plans around the crown target shown in the example, but updates, maps, or task rolls can change how a run is judged.

### What is the best first map or difficulty?

Start with the smallest / easiest setup available, such as Cottage on Easy if your build offers it. Learn safe routes before moving into larger maps.

### What controls should I remember first?

Remember **Tab** for the task list, **E** for interact / untie, and **Q** to drop held items. These three controls fix many beginner mistakes.

### Is sprinting bad?

Sprinting is useful for escape, but bad for scouting. It burns stamina and creates more noise, so use it when you actually need speed.

### How do I know which room a task item belongs to?

Use the task wording first. If it says kids room, dining room, kitchen, bathroom, or hallway, confirm the room before carrying objects outside. If the task does not update, stop testing random items and finish a safer objective.

### Can I play Burglin' Gnomes solo?

You can attempt runs solo, but the game is much easier with friends because rescue, lookout, carrying, and distraction can be split between players.
